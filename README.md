# AWS Cloud Security Posture Assessment Lab

Self-directed cloud security lab built on AWS Free Tier to simulate and remediate real-world cloud misconfigurations.

## What I did
- Created intentional misconfigurations across IAM, S3, and VPC
- Used AWS Security Hub (CSPM) to detect 88+ findings
- Remediated all High and Medium severity findings
- Documented findings in a formal security assessment report

## Tools Used
AWS Security Hub, IAM, S3, VPC, EC2
